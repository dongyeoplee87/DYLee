## 💡 학습 팁

### 문제 풀이 전략
1. **5분 규칙**: 5분 고민 후 접근법 떠오르지 않으면 힌트 확인
2. **30분 규칙**: 30분 이상 막히면 솔루션 참고 후 재구현
3. **복습 주기**: 1일 후, 3일 후, 1주일 후 재풀이

### 코드 작성 습관
- 변수명 명확하게 작성
- 엣지 케이스 미리 생각 (빈 배열, 최대/최소값 등)
- 시간복잡도 주석으로 표기

### 디버깅 팁
- 작은 테스트 케이스부터 검증
- print 디버깅보다 논리적 추론 우선
- 경계값 테스트 필수

### 모의고사 시뮬레이션
- 실제 시험 환경 조성 (시간 제한, 인터넷 제한)
- 코드 제출 전 반드시 재검토
- 부분 점수 전략 수립
